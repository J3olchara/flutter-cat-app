# Релизы Кототиндер


[Github releases](https://github.com/J3olchara/flutter-cat-app/releases)

### Технические детали:
- Flutter SDK: 3.10.3+
- Минимальная версия Android: API 21 (Android 5.0)
- Размер APK: 50.3 MB

### Системные требования:
- Android 5.0 (Lollipop) или выше
- Подключение к интернету для загрузки данных

### Установка:
1. Скачайте APK файл
2. Разрешите установку из неизвестных источников в настройках
3. Откройте APK и следуйте инструкциям установки
